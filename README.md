# 💳 Credit Card Fraud Detection Using Random Forest

## 📌 Overview

This project is a Machine Learning-based Credit Card Fraud Detection system developed using Python and Streamlit. The application predicts whether a credit card transaction is **Valid (0)** or **Fraudulent (1)** using a Random Forest Classifier.

The system includes data preprocessing, handling of imbalanced datasets, model training, evaluation, visualization, and real-time prediction through an interactive web interface.

---

## 🚀 Features

* Upload custom training datasets in CSV format
* Automatic preprocessing of missing and categorical data
* Handles class imbalance using Random Oversampling
* Trains a Random Forest Classifier
* Displays model accuracy and classification report
* Visualizes confusion matrix
* Shows feature importance analysis
* Generates correlation heatmaps
* Predicts fraudulent transactions on uploaded test datasets
* Download prediction results as CSV

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Thanujaroyal24/creditcard-fraud-detection.git
```

Navigate to the project directory:

```bash
cd creditcard-fraud-detection
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
